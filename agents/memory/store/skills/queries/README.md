# Skills Queries

This directory holds generated or curated indices for procedural memory.

Candidate indices:

- `by-domain.md`
- `by-trigger.md`
- `by-wiki-page.md`
- `by-version.md`
- `promotion-candidates.md`

Indices are navigation aids. `SKILL.md`, learning records, and evidence refs remain the source of truth.

Ad hoc search:

```bash
python3 agents/memory/store/skills/scripts/query_skills.py "global norm" --source all
```
