# Wiki Data

This directory stores schema notes and maintenance metadata for the LLM Wiki.

Current roles:

- `schema.md`: required frontmatter and optional KDA-inspired metadata.
- `freshness.md`: how to record source cutoff and version-sensitive claims.
- `provenance.md`: how to cite workspace evidence, task ids, and source documents.

These files guide Memory Agent and Starter Agent behavior. They are not task evidence by themselves.
